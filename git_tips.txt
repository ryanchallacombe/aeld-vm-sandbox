Set the new branch:
Run the following command from the root of your main repository:
git submodule set-branch -b <branch_name> <submodule_path>
Sync and update:
Apply the configuration change to your local environment:
git submodule sync
git submodule update --remote --init
Commit the change:
You must commit the changes to the .gitmodules file in your main repository:
git add .gitmodules <submodule_path>
git commit -m "Switch submodule to <branch_name> branch"
